# CICD_Pipeline

# Overview

This project demonstrates a CI/CD pipeline for automating machine learning model training using GitHub Actions. The pipeline ensures that the model is automatically trained and validated whenever code changes are pushed.


# Tech Stack

Python, GitHub Actions, Pandas, Scikit-learn


# Workflow
Code push triggers CI/CD pipeline
Dependencies are installed
Model training script (train_model.py) runs
Dataset (iris.csv) is processed
Model is trained and validated automatically

# Files
train_model.py – Model training script
iris.csv – Dataset
requirements.txt – Dependencies
.github/workflows/ – CI/CD configuration

# Features
Automated model training
Continuous integration using GitHub Actions
Simple and reproducible ML workflow

# Conclusion

A lightweight CI/CD implementation for automating machine learning pipelines.
